Source docstring:

```
Returns the nth rest of coll, coll when n is 0.
```

Source code:

```clj
(defn nthrest
  [coll n]
    (loop [n n xs coll]
      (if (and (pos? n) (seq xs))
        (recur (dec n) (rest xs))
        xs)))
```
