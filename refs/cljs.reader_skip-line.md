Source docstring:

```
Advances the reader to the end of a line. Returns the reader
```

Source code:

```clj
(defn skip-line
  [reader _]
  (loop []
    (let [ch (read-char reader)]
      (if (or (identical? ch \newline) (identical? ch \return) (nil? ch))
        reader
        (recur)))))
```
