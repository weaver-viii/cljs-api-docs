Source docstring:

```
Evaluates expr and prints the time it took. Returns the value of expr.
```

Source code:

```clj
(defmacro time
  [expr]
  `(let [start# (.getTime (js/Date.))
         ret# ~expr]
     (prn (core/str "Elapsed time: " (- (.getTime (js/Date.)) start#) " msecs"))
     ret#))
```
