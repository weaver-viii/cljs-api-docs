Returns a map with `keys` mapped to corresponding `vals`.

<pre>user=> (zipmap [:a :b :c :d] [1 2 3 4])
{:a 1, :b 2, :c 3, :d 4}</pre>

Source docstring:

```
Returns a map with the keys mapped to the corresponding vals.
```

Source code:

```clj
(defn zipmap
  [keys vals]
    (loop [map (transient {})
           ks (seq keys)
           vs (seq vals)]
      (if (and ks vs)
        (recur (assoc! map (first ks) (first vs))
               (next ks)
               (next vs))
        (persistent! map))))
```
