 <samp>
(__bit-set__ x n)<br>
</samp>

---

Set bit at index `n`.  Same as `x | (1 << y)` in JavaScript.

---

Example:

Bits can be entered using radix notation:

```clj
(bit-set 2r1100 1)
;;=> 14
;; 14 = 2r1110
```

Same number in decimal:

```clj
(bit-set 12 1)
;;=> 14
```
