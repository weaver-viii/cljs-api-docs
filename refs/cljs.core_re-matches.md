---



Source docstring:

```
Returns the result of (re-find re s) if re fully matches s.
```

Source code:

```clj
(defn re-matches
  [re s]
  (let [matches (.exec re s)]
    (when (= (first matches) s)
      (if (== (count matches) 1)
        (first matches)
        (vec matches)))))
```
