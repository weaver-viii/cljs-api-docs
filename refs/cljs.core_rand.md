Returns a random floating point number between 0 inclusive and `n` exclusive.

`n` defaults to 1.

Source docstring:

```
Returns a random floating point number between 0 (inclusive) and
n (default 1) (exclusive).
```

Source code:

```clj
(defn rand
  ([] (rand 1))
  ([n] (* (Math/random) n)))
```
