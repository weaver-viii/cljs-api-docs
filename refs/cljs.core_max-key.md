Returns the `x` for which `(k x)` is greatest.

`(k x)` should return a number.

Source docstring:

```
Returns the x for which (k x), a number, is greatest.
```

Source code:

```clj
(defn max-key
  ([k x] x)
  ([k x y] (if (> (k x) (k y)) x y))
  ([k x y & more]
   (reduce #(max-key k %1 %2) (max-key k x y) more)))
```
