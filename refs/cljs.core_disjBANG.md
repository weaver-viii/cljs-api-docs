```
disj[oin]. Returns a transient set of the same (hashed/sorted) type, that

```

Source code:

```clj
(defn disj!
  ([tcoll val]
    (-disjoin! tcoll val))
  ([tcoll val & vals]
    (let [ntcoll (-disjoin! tcoll val)]
      (if vals
        (recur ntcoll (first vals) (next vals))
        ntcoll))))
```
