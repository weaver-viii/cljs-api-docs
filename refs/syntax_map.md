Signifies a map.  Must contain pairs of key-values.  Keys and values can be any type.

---

Example:

```clj
{:foo 1 :bar 2}
;;=> {:foo 1, :bar 2}
```

Use any value as a key:

```clj
(def m {[1 2] 3})
(get m [1 2])
;;=> 3
```
