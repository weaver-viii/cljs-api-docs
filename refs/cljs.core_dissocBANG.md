Source docstring:

```
Returns a transient map that doesn't contain a mapping for key(s).
```

Source code:

```clj
(defn dissoc!
  ([tcoll key]
    (-dissoc! tcoll key))
  ([tcoll key & ks]
    (let [ntcoll (-dissoc! tcoll key)]
      (if ks
        (recur ntcoll (first ks) (next ks))
        ntcoll))))
```
