```
Returns true if (pred x) is logical true for every x in coll, else

```

Source code:

```clj
(defn ^boolean every?
  [pred coll]
  (cond
   (nil? (seq coll)) true
   (pred (first coll)) (recur pred (next coll))
   :else false))
```
