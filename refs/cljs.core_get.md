---

Returns the value mapped to key `k`.

Returns `not-found` or nil if `k` is not present in `o`.

---

Source docstring:

```
Returns the value mapped to key, not-found or nil if key not present.
```

Source code:

```clj
(defn get
  ([o k]
     (-lookup o k))
  ([o k not-found]
     (-lookup o k not-found)))
```
