 <samp>
(__bit-flip__ x n)<br>
</samp>

---

Flip bit at index `n`.  Same as `x ^ (1 << y)` in JavaScript.

---

Example:

Bits can be entered using radix notation:

```clj
(bit-flip 2r1111 2)
;;=> 11
;; 11 = 2r1011
```

Same numbers in decimal:

```clj
(bit-flip 15 2)
;;=> 11
```
