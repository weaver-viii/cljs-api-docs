 <samp>
(__subset?__ a b)<br>
</samp>

---

Returns true if `a` is a subset of `b`, false otherwise.

In other words, returns true if all the elements of `a` can be found in `b`.

Source docstring:

```
Is set1 a subset of set2?
```

Source code:

```clj
(defn subset? 
  [set1 set2]
  (and (<= (count set1) (count set2))
       (every? #(contains? set2 %) set1)))
```
