Returns the `n`th `next` of `coll`.

Returns `(seq coll)` when `n` is 0.

Source docstring:

```
Returns the nth next of coll, (seq coll) when n is 0.
```

Source code:

```clj
(defn nthnext
  [coll n]
  (loop [n n xs (seq coll)]
    (if (and xs (pos? n))
      (recur (dec n) (next xs))
      xs)))
```
