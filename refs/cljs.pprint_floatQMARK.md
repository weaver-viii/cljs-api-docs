Source docstring:

```
Returns true if n is an float.
```

Source code:

```clj
(defn ^boolean float?
  [n]
  (and (number? n)
       (not ^boolean (js/isNaN n))
       (not (identical? n js/Infinity))
       (not (== (js/parseFloat n) (js/parseInt n 10)))))
```
