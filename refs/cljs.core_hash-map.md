Source code:

```clj
(defn hash-map
  [& keyvals]
  (loop [in (seq keyvals), out (transient (.-EMPTY PersistentHashMap))]
    (if in
      (recur (nnext in) (assoc! out (first in) (second in)))
      (persistent! out))))
```

(defmacro hash-map

    (let [pairs (partition 2 kvs)
          ks    (map first pairs)
          vs    (map second pairs)]
