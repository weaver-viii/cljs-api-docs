Returns the namespace String of a symbol or keyword, or nil if not present.

Source code:

```clj
(defn namespace
  [x]
  (if (implements? INamed x)
    (-namespace ^not-native x)
    (throw (js/Error. (str "Doesn't support namespace: " x)))))
```
