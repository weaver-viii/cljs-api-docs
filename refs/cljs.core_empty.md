 <samp>
(__empty__ coll)<br>
</samp>

---

Returns an empty collection of the same category as `coll`.

Returns nil if `coll` is nil.

---


See Also:

[`cljs.core/not-empty`](cljs.core_not-empty.md)<br>

---

Source docstring:

```
Returns an empty collection of the same category as coll, or nil
```

Source code:

```clj
(defn empty
  [coll]
  (when-not (nil? coll)
    (-empty coll)))
```
