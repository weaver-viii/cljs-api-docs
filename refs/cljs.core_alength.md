(__alength__ a)<br>
</samp>

---

For interop, it returns the length of a JavaScript array or string.

---

Example:

```clj
(def a #js [1 2 3])

(alength a)
;;=> 3

(.-length a)
;;=> 3

(aget a "length")
;;=> 3

(count a)
;;=> 3
```

---

See Also:

[`cljs.core/count`](cljs.core_count.md)<br>

---

Source docstring:

```
Returns the length of the array. Works on arrays of all types.
```
