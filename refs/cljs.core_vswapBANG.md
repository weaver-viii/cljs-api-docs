Source docstring:

```
Non-atomically swaps the value of the volatile as if:
(apply f current-value-of-vol args). Returns the value that
was swapped in.
```

Source code:

```clj
(defmacro vswap!
  [vol f & args]
  `(-vreset! ~vol (~f (-deref ~vol) ~@args)))
```
