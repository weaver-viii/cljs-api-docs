Return a lazy sequence of all but the last `n` items in `s`.

`n` defaults to 1.

---

```
Return a lazy sequence of all but the last n (default 1) items in coll
```

Source code:

```clj
(defn drop-last
  ([s] (drop-last 1 s))
  ([n s] (map (fn [x _] x) s (drop n s))))
```
