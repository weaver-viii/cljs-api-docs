(__array-seq__ array i)<br>

---

Creates a `seq` from a JavaScript array, starting at index `i` if given.

---

Example:

```clj
(array-seq #js [1 2 3])
;;=> (1 2 3)

(array-seq #js [1 2 3] 1)
;;=> (2 3)
```

---



Source code:

```clj
(defn array-seq
  ([array]
     (prim-seq array 0))
  ([array i]
     (prim-seq array i)))
```
