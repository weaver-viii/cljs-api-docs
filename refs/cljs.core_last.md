 <samp>
(__last__ coll)<br>
</samp>

---

Returns the last item in `coll` in linear time.

`peek` is much faster than `last` for a vector.

---

Example:

```clj
(last [1 2 3])
;;=> 3

(last [1 2])
;;=> 2

(last [1])
;;=> 1

(last [])
;;=> nil
```

```
Return the last item in coll, in linear time
```

Source code:

```clj
(defn last
  [s]
  (let [sn (next s)]
    (if-not (nil? sn)
      (recur sn)
      (first s))))
```
