 <samp>
(__dir__ ns)<br>
</samp>

---

Prints a sorted directory of public vars in a namespace `ns`.

---

Example:

```clj
(require 'clojure.set)

(dir clojure.set)
;; Prints:
;;  difference
;;  index
;;  intersection
;;  join
;;  map-invert
;;  project
;;  rename
;;  rename-keys
;;  select
;;  subset?
;;  superset?
;;  union
;;
;;=> nil
```

(defmacro dir
  [ns]
  `(doseq [sym# (quote ~(sort (keys (ana-api/ns-publics ns))))]
     (println sym#)))
