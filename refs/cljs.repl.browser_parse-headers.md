```
Parse the headers of an HTTP POST request.
```

Source code:

```clj
(defn parse-headers
  [header-lines]
  (apply hash-map
   (mapcat
    (fn [line]
      (let [[k v] (str/split line #":" 2)]
        [(keyword (str/lower-case k)) (str/triml v)]))
    header-lines)))
```
