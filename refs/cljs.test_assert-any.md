```
Returns generic assertion code for any test, including macros, Java
method calls, or isolated symbols.
```

Source code:

```clj
(defn assert-any
  [msg form]
  `(let [value# ~form]
     (if value#
       (cljs.test/do-report
         {:type :pass, :message ~msg,
          :expected '~form, :actual value#})
       (cljs.test/do-report
         {:type :fail, :message ~msg,
          :expected '~form, :actual value#}))
     value#))
```
