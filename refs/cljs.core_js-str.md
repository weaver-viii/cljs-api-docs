 <samp>
(__js-str__ s)<br>
</samp>

---

Convert `s` to string using JavaScript's coercion behavior.

Equivalent to `''+s` in JavaScript.

---

Example:

```clj
(js-str 23)
;;=> "23"

(js-str #js {:foo 1})
;;=> "[Object object]"
```

  (core/list 'js* "''+~{}" s))
