Returns a lazy sequence of `x`s.

The length of the sequence is infinite, or `n` if provided.

Source docstring:

```
Returns a lazy (infinite!, or length n if supplied) sequence of xs.
```

Source code:

```clj
(defn repeat
  ([x] (lazy-seq (cons x (repeat x))))
  ([n x] (take n (repeat x))))
```
