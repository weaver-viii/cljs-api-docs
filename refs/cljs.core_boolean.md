 <samp>
(__boolean__ x)<br>
</samp>

---

Return `false` if `x` is false or nil.  Otherwise return `true`.  This is the
truthiness condition used by `if` expressions.

---

Example:

```clj
(boolean 1)
;;=> true

(boolean 0)
;;=> true

(boolean nil)
;;=> false
```

---

See Also:

[`special/if`](special_if.md)<br>

---


Source code:

```clj
(defn ^boolean boolean [x]
  (if x true false))
```
