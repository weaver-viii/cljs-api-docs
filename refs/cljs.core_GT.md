---

Returns true if each successive number argument is less than the previous
one, false otherwise.

---

Example:

```clj
(> 1 2)
;;=> false

(> 2 1)
;;=> true

(> 2 2)
;;=> false

(> 6 5 4 3 2)
;;=> true
```

Source docstring:

```
Returns non-nil if nums are in monotonically decreasing order,
otherwise false.
```

Source code:

```clj
(defn ^boolean >
  ([x] true)
  ([x y] (cljs.core/> x y))
  ([x y & more]
   (if (cljs.core/> x y)
     (if (next more)
       (recur y (first more) (next more))
       (cljs.core/> y (first more)))
     false)))
```
