Source docstring:

```
Coerces coll to a (possibly empty) sequence, if it is not already
one. Will not force a lazy seq. (sequence nil) yields ()
```

Source code:

```clj
(defn ^seq sequence
  [coll]
   (if (seq? coll)
     coll
     (or (seq coll) ())))
```
