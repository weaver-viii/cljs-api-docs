Source docstring:

```
Evaluates body with *read-eval* set to a "known" value,
i.e. substituting true for :unknown if necessary.
```

Source code:

```clj
(defmacro with-read-known
  [& body]
  `(binding [*read-eval* (if (= :unknown *read-eval*) true *read-eval*)]
     ~@body))
```
