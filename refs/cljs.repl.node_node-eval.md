```
Evaluate a JavaScript string in the Node REPL process.
```

Source code:

```clj
(defn node-eval
  [repl-env js]
  (let [{:keys [in out]} @(:socket repl-env)]
    (write out js)
    (let [result (json/read-str
                   (read-response in) :key-fn keyword)]
      (condp = (:status result)
        "success"
        {:status :success
         :value (:value result)}

        "exception"
        {:status :exception
         :value (:value result)}))))
```
