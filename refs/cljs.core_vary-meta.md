 <samp>
(__vary-meta__ obj f & args)<br>
</samp>

---

Returns an object of the same type and value as `obj`, with
`(apply f (meta obj) args)` as its metadata.

---

Example:

```clj
(def a ^:foo [1 2 3])
(def b (vary-meta a assoc :bar true))

(= a b)
;;=> true

(meta a)
;;=> {:foo true}

(meta b)
;;=> {:foo true, :bar true}
```
