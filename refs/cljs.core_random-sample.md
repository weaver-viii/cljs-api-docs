Source docstring:

```
Returns items from coll with random probability of prob (0.0 -
1.0).  Returns a transducer when no collection is provided.
```

Source code:

```clj
(defn random-sample
  ([prob]
     (filter (fn [_] (< (rand) prob))))
  ([prob coll]
     (filter (fn [_] (< (rand) prob)) coll)))
```
