 <samp>
(__associative?__ coll)<br>
</samp>

---

Returns true if `coll` implements the `IAssociative` protocol, false otherwise.

Maps and vectors are associative.

---

Example:

```clj
(associative? [1 2 3])
;;=> true

(associative? {:a 1 :b 2})
;;=> true

(associative? #{1 2 3})
;;=> false

(associative? '(1 2 3))
;;=> false
```

---

