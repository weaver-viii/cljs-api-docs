Source docstring:

```
Efficient test to determine that two keywords are identical.
```

Source code:

```clj
(defn ^boolean keyword-identical?
  [x y]
  (if (identical? x y)
    true
    (if (and (keyword? x)
             (keyword? y))
      (identical? (.-fqn x) (.-fqn y))
      false)))
```
