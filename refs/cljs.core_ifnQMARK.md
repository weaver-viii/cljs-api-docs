 <samp>
(__ifn?__ f)<br>
</samp>

---

Returns true if `f` implements the `IFn` protocol, false otherwise.

Functions, keywords, map, sets, and vectors can be called as functions.

---


See Also:

[`cljs.core/fn?`](cljs.core_fnQMARK.md)<br>

---


Source code:

```clj
(defn ^boolean ifn? [f]
  (or (fn? f) (satisfies? IFn f)))
```
