(__source__ name)<br>
</samp>

---

Prints the source code for the given symbol `name`, if it can find it.  This
requires that the symbol resolve to a Var defined in a namespace for which the
.cljs is in the classpath.

---

Example:

```clj
(source comment)
;; Prints:
;; (defmacro comment
;;   "Ignores body, yields nil"
;;   {:added "1.0"}
;;   [& body])
;;
;;=> nil
```
